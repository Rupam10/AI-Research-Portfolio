# Paper 1

## Error Analysis of a CNN-Based Tomato Leaf Disease Classification Model

This paper presents an independent evaluation of a CNN model trained for tomato leaf disease classification.

The research focuses on:

- Confusion Matrix Analysis
- Class-wise Precision, Recall and F1 Score
- Model Weakness Identification
- Independent Real-world Validation
- Error Analysis
- Practical Improvement Recommendations

## Research Contribution

Instead of reporting only overall accuracy, this study investigates where the model fails and why those failures matter for real-world agricultural deployment.
